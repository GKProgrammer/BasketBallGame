# BasketBall Game Overview
This is a personal project I've built using Unity 2D,
- The game has a total of 6 levels which get harder and harder testing your pacience.
- As the game's name suggests, you are a basketball, trying to go past the hurdles you encounter while trying to escape.
- Your objective is to collect all the coins and open locked doors using the buttons.
- You won't be able to finish the level if you do not collect all the coins.
- These objects instantly kill you: Spikes, Saws and Turrets

# Minimum Requirements (Windows)
- OS Version: Windows 10 version 21H1 (build 19043) or newer
- CPU: x86, x64 architecture with SSE2 instruction set support, Arm64
- Graphics API: DX10, DX11, DX12 or Vulkan capable GPUs
- RAM: 2GB+
- Storage Space: 93 MB

# Minimum Requirements (Android)
- OS Version: 6.0 (API 23)+ Customized versions of Android must include all supported Google standard APIs.
- CPU: ARMv7 with Neon Support (32-bit) or ARM64
- Graphics API: OpenGL ES 3.0+, Vulkan
- RAM: 1GB+
- Storage Space: 120 MB
